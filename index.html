O projeto tem como objetivo o desenvolvimento de uma plataforma web interativa para análise e visualização de dados meteorológicos. A aplicação será baseada em um banco de dados MongoDB, responsável por armazenar dados como temperatura, umidade, precipitação, velocidade do vento, entre outros. A camada de visualização será composta por dashboards interativos desenvolvidos com o uso do Metabase, além de integração com páginas web construídas em HTML, CSS e Java, proporcionando uma experiência acessível e intuitiva. O projeto também contemplará a criação de aproximadamente cinco dashboards, abordando diferentes perspectivas dos dados, como: Análise temporal (variação ao longo do tempo) Comparações entre regiões Identificação de padrões climáticos Indicadores extremos (eventos fora do padrão) Tendências e previsões simples
